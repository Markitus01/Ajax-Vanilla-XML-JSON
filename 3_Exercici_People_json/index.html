Dins el servidor existeix l’arxiu people.json i fields.json.

  • Al carregar la pàgina mostrar dins l'etiqueta amb id fields un conjunt de radio buttons amb els valors de l’array que hi ha en l’arxiu fields.json.
  • Quan l’usuari marqui un radio button s’ha de mostrar una taula dins l'etiqueta amb id dades amb dues columnes:
    1. un checkbox per cada fila.
    2. valors sense repeticions que existeixen dins l’arxiu people.json del camp seleccionat, ordenats alfabeticament.
    L’usuari pot seleccionar tants checkbox com desitgi.
  • Al premer el botó es visualitzaran totes les dades de les persones que compleixin amb el/s valor/s seleccionats dins l'etiqueta amb id persones. Es pot escollir entre 2 comportaments pel botó:
    1. Desactivat/Ocult per defecte i només s'activa/visualitza quan hi hagi un checkbox seleccionat.
    2. Avisar a l'usuari si prem el botó i no hi ha cap checkbox seleccionat.
